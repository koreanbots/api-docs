# 유저

<h2 id="user">유저 정보</h2>

### 구조

| FIELD | TYPE | DESCRIPTION |
|-------|------|-------------|
| id | string | 유저의 ID |
| username | string | 유저의 디스코드 사용자 이름 |
| tag | string | 유저의 디스코드 태그 |
| github | ?string | 깃허브 닉네임 |
| flags | integer | 유저의 [플래그](#flags) |
| bots | Bot[] | 소유한 봇들 (단, 소유자는 아이디만 표시됩니다) |

<h2 id="flags">유저 플래그(flags)</h2>

| VALUE | DESCRIPTION |
|-------|-------------|
| 0 << 0 | 없음 |
| 1 << 0 | 관리자 |
| 1 << 1 | 버그 헌터 |
| 1 << 2 | 봇 리뷰어 |
| 1 << 3 | 프리미엄 유저 |
