# 위젯

## 봇 위젯 가져오기

### `GET` /widget/bots/{widget.type}/{bot.id}.svg

### widget.type
- `votes` 하트 수 위젯
- `servers` 서버 수 위젯
- `status` 봇 상태 위젯

### Query String

| FIELD | TYPE | DESCRIPTION | REQUIRED | DEFAULT |
|-------|------|-------------|----------|---------|
| style | `classic` 또는 `flat` | 위젯 타입 | false | flat |
| scale | number | 위젯 사이즈 (0.5~3.0) | false | 1.0 |
| icon | boolean | 봇 이미지 표시 여부 | false | true |
